## Commands
### run
```
]run language
code
```
runs your code.

And the bot ignores "```" for a code block.
#### Example<a id="e1"></a>
```
]run python
print('hello')
```

### save
```
]save language
code
```
saves your code.
You can call it by sending as follows.
```
]run saved
standard input
```

### tex
```
]tex
math commands
```
images your math commands.

And the bot ignores "```" for a code block.

It uses LaTeX gather* environment.
#### Example<a id="e2"></a>
```
]tex
\int_0^1 f(x)\,dx
```
### texp
```
]texp
LaTeX text
